En esta política de privacidad se especificará todo lo relacionado a los datos que la aplicación "Helpin" podría acceder o pedir al usuario, así como los que el usuario podría introducir por su propia cuenta

 Datos a los que se podría acceder: La aplicación "Helpin" fue desarrollada mediante el uso de la tecnología React Native (específicamente para el sistema operativo Android, aunque próximamente para IOS). Esto se hace mención, debido a que en ningún momento los desarrolladores de "Helpin" acceden a datos personales del usuario. Sin embargo, debido a que en React Native hay que instalar varios modulos de terceros para incorporar ciertas funcionalidades (como, por mencionar un ejemplo, en la seccion "Practica tu Speaking" donde se involucró un módulo de tercero para el componente de micrófono), desconocemos si estos podrían acceder a información personal del teléfono usuario. Por parte del equipo de desarrollo de "Helpin", no se ha escrito conscientemente alguna línea de código que pudiese acceder a algún sistema de almacenamiento, dado que la aplicación no lo necesita. Por parte de los modulos de terceros, se está en un parcial desconocimiento.

 Datos que se podrían pedir, o que se podrían introducir sin petición por la aplicación: Solo hay tres tipos de datos que el usuario podría introducir a la interfaz "Helpin"
1.- Gestos de cualquier GUI (Interfaz gráfica de usuario): este tipo de información solamente involucra desplazamientos con el dedo, así como toques en botones que no hacen nada más que mostrar y extender la información que se quiere comunicar (en este caso, información para aprender temas de Inglés). En ningún momento se está insertando información personal, únicamente gestos para movilidad dentro de la app.
2.- Respuestas en quices: solamente involucra que el usuario seleccione la respuesta correcta, dentro de un grupo de respuestas. En ningún momento se está insertando información personal, únicamente se está eligiendo información que la misma aplicación propone.
